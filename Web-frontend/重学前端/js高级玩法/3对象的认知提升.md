## 为什么设置常规属性和排序属性
- 提高访问速度
- 两种线性数据结构保存
  - elements 排序属性
  - properties 常规属性
  - 内置属性 10个以内，多余会放到properties里面
- 不破坏有序属性 
- 隐藏类
  - 从空间时间维度进行提升性能
- 守护隐藏类
  - 初始化属性顺序一致 
  - 一次性声明完毕
  - 谨慎使用delete
